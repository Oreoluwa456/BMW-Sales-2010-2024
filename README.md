
BMW Sales Dashboard (2010–2024)

An interactive Microsoft Excel dashboard that analyzes BMW sales data from 2010 to 2024. Built using PivotTables, PivotCharts, Slicers, and Excel features, the dashboard provides a comprehensive view of sales performance, regional distribution, customer preferences, and vehicle specifications to support data-driven decision-making.

Project Objectives

- Analyze BMW sales performance from 2010 to 2024.
- Monitor key business metrics in a single dashboard.
- Compare sales performance across different regions.
- Identify the best-selling BMW models.
- Analyze customer preferences by fuel type, transmission, and vehicle color.
- Track yearly sales trends to identify performance patterns.

Key Insights
- Asia recorded the highest BMW sales volume, followed by Europe and North America.
- The BMW 7 Series was the best-selling model, followed closely by the i8, X1, 3 Series, and i3.
- Hybrid vehicles achieved the highest sales quantity, while demand across all fuel types remained relatively balanced.
- Customer preference for Automatic and Manual transmissions was nearly equal.
- Red, Silver, and White were among the most popular vehicle colors.
- Sales remained relatively stable throughout the period, with stronger performance observed between 2022 and 2024.

Conclusion
The BMW Sales Dashboard provides a comprehensive analysis of sales performance from 2010 to 2024 using Microsoft Excel. By consolidating key metrics and interactive visualizations into a single dashboard, it enables users to monitor sales trends, compare regional performance, evaluate customer preferences, and identify top-performing vehicle models. This project demonstrates how Excel can be effectively used to transform raw sales data into meaningful insights that support informed business decisions.
